hash

module documentation

Calculates the hash value of the input string x using the polynomial hash function. Args: x (str): The input string for which the hash value is calculated. Returns: str: The hash value in the form of a hexadecimal string. The hash function calculates the hash value using the formula: hash(x) = (x[0] * P) ** (Q + x[0]) + (x[1] * P) ** (Q + x[1]) + ... Where: M (int): The big prime number used for modulo in the calculation. P (int): The prime number used as the coefficient for x. Q (int): The prime number used as the degree indicator in the polynomial.
