class Node(pygame.sprite.Sprite):

Constructor: Node(pos, status, icon_speed, path)

View In Hierarchy

Represents a node on the Overworld map. Attributes: - frames (list): A list of images for animation. - frame_index (float): Index representing the current frame in the animation. - image (pygame.Surface): The current image of the node. - status (str): The status of the node ('available' or 'locked'). - rect (pygame.Rect): The rectangular area occupied by the node on the screen. - detection_zone (pygame.Rect): The collision detection zone around the node. Methods: - __init__(self, pos, status, icon_speed, path): Initializes a new Node instance. - animate(self): Animates the node by updating the frame index and image. - update(self): Updates the node, applying animations and handling locked status tinting.
